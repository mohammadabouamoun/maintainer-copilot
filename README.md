# Maintainer's Copilot

An AI-powered assistant for open-source maintainers to triage issues, classify, retrieve context, and answer questions with long-term memory.

Built for Week 7 project.

## Repository

`fastapi/fastapi` issues dataset

## Stack

- FastAPI backend
- PostgreSQL + pgvector
- Redis
- MinIO
- Vault
- React widget
- Streamlit admin
