Testing & Configuration Tips

You can use the floating widget in the bottom-right corner to interact with the Maintainer's Copilot. Here are a few queries you can try:

  • Ask for Pandas 2.0.0 changes: "Search the documentation for What's new in pandas 2.0.0"
  • Triage an issue: "Classify this issue: When importing pandas 2.0 on python 3.12, the interpreter throws an ImportError."
  • Teach the bot: "Remember that I am using Ubuntu 22.04 and prefer dark mode." then ask "What is my preferred OS?"
